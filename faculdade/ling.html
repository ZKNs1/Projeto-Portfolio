Variáveis

    String nome = "Melina";

Operadores

São usados para realizar operações com valores ou variáveis, eles permitem a manipulação de dados, execução de cálculos, e controle do fluxo de execução.

Operados Aritméticos

São usados para realizar operações matemáticas básicas.

    int a = 3 + 5; // Soma
    int b = 23 - 12; // Subtração;
    int c = 7 * 4 // Multiplicação;
    int d = 6 / 2 // Divisão;
    int f = 15 % 4;  // Resto;

Operados Comparação

São usados para comparar dois valores e retornar um valor booleano.

    a == b; // a igual a b
    a != b; // a diferente de b
    a > b; // a  maior que b
    a < b; // a menor que b
    a >= b; // a maior ou igual a b
    a <= b; // a menor ou igual a b

Operados Lógica

São usados para combinar expressões booleanas.

    (a && b) // 'E' = a e b
    (a || b) // 'OU' = a ou b
    result = (a > b) && (a < c);  // Retorna true se ambos a > b e a < c forem true
    result = (a > b) || (a < c);  // Retorna true se pelo menos uma entre a > b ou a < c for true
    result = !(a > b);  // Retorna true se a > b for false

Estrutura de decisão

Utilizada para tomar decisões com base em condições.

If/Else

O if e o else se complementam, de forma que se algo não acontecer (if) o programa deverá tomar outra atitude(else).

    int a = 5;
    int b = 3;

    //Compara o valor de 'a' e de 'b'

    if(a>b){ //Caso 'a' seja maior que 'b'
        System.out.println("'a' é maior que 'b")
    }

    else{ //Caso 'b' for maior que 'a'
        System.out.println("'b' é maior que 'a'")
    }

Switch

O switch é utilizado para selecionar um bloco de código a ser executado dentre várias opções, com base no valor de uma expressão.

    //O sistema deverá responder com o respectivo dia, neste caso deverá executar o caso 3 e dizer que é terça-feira.
    int day = 3;
  
    switch (day) {
        case 1:
            System.out.println("Domingo");
            break;
        case 2:
            System.out.println("Segunda-feira");
            break;
        case 3:
            System.out.println("Terça-feira");
            break;
        case 4:
            System.out.println("Quarta-feira");
            break;
        case 5:
            System.out.println("Quinta-feira");
            break;
        case 6:
            System.out.println("Sexta-feira");
            break;
        case 7:
            System.out.println("Sábado");
            break;
        default:
            System.out.println("Dia inválido");
            break;
    }

Estruturas Repetição

São usados para realizar a mesma operação diversas vezes.

For

Repete um pedaço do código por um número específico de vezes. O “for” inicializa a variável i, depois verifica se seu valor é menor que outro, e incrementa i a cada repetição, o loop encerra quando o valor de i passar o valor específicado.

    for (int i = 0; i < 10; i++) {
        System.out.println("Valor de i: " + i);
    }

While

Executa um bloco de instruções repetidas vezes enquanto uma condição for verdadeira.

    int i = 0;
    while (i < 10) {
        System.out.println("Valor de i: " + i);
        i++;
    }

Vetores

Os vetores permitem o armazenamento de uma coleção de valores do mesmo tipo e são unidimensionais.

    int[] numeros = {1, 2, 3, 4, 5};

    System.out.println(numeros[0]); // Imprime o primeiro elemento: 1

Matrizes

As matrizes são usadas para representar uma certa quantidade de variáveis de valores homogêneos e armazena elementos em grade bidimensional ou superior.

    int[][] matriz = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };

    System.out.println(matriz[0][1]); // Imprime o elemento na primeira linha, segunda coluna: 2

Funções

Função é um bloco de código que executa alguma operação, podem definir parâmetros de entrada permitindo chama-la e passar argumentos, além de também poder retornar um valor como saída. 

    // A função “areaCirculo” é uma forma de encapsular a lógica de calcular a área de um círculo em um único bloco de código reutilizável.
 
    // Definindo a função areaCirculo
    public static double areaCirculo(double raio) {
        double area = Math.PI * raio * raio;
        return area;
    }

    public static void main(String[] args) {
        // Chamando a função areaCirculo e armazenando o resultado
        double raio = 5.0;
        double area = areaCirculo(raio);

        System.out.println("A área de um círculo com raio " + raio + " é: " + area);
    }

POO

Programação Orientada a Objetos
A programação orientada a objetos organiza o software em torno de "objetos" em vez de funções e lógica.
Um objeto é uma instância de uma classe, ou seja, é uma estrutura que define as propriedades chamados de atributos e comportamentos chamados de métodos que o objeto pode ter.

Classes e objetos
Uma classe é uma forma de definir um tipo de dado em uma linguagem orientada a objeto. Ela é formada por dados (atributos) e comportamentos (métodos)
Objeto é uma instância de uma classe. Cada objeto pode ter diferentes valores para os atributos definidos na classe. 

    public class Pessoa {
        // Atributos da classe
        private String nome;
        private int idade;
    
        // Construtor da classe
        public Pessoa(String nome, int idade) {
            this.nome = nome;
            this.idade = idade;
        }
    
        // Método da classe
        public void apresentar() {
            System.out.println("Oi, eu sou  " + nome + " e tenho " + idade + " anos.");
        }
    }


    public class Main {
        public static void main(String[] args) {
            // Criação de um objeto da classe Pessoa
            Pessoa pessoa = new Pessoa("João", 25);
            // Chamada do método apresentar do objeto pessoa
            pessoa.apresentar(); // Imprime: Oi, eu sou João e tenho 25 anos.
        }
    }

Encapsulamento
O encapsulamento é uma forma de esconder os detalhes internos de um objeto e permitir o acesso apenas através de métodos públicos. 

    public class Pessoa {
        // Apenas acessíveis nesta classe
        private String nome;
        private int idade;
    
        //Acessíveis em todas as classes
        // Métodos getter e setter para o atributo nome
        public String getNome() {
            return nome;
        }
    
        public void setNome(String nome) {
            this.nome = nome;
        }
    
        // Métodos getter e setter para o atributo idade
        public int getIdade() {
            return idade;
        }
    
        public void setIdade(int idade) {
            this.idade = idade;
        }
    }
