Programação:

A programação é feita utilizando as linguagens HTMl, CSS e JavaScript, junto do software Visual Studio Code.

Clique aqui para acessar o repositório no GitHub

HTML

HyperText Markup Language

Nós utilizamos o HTML para a criação de páginas web, permitindo estruturar todo o conteúdo de uma página utilizando marcação, para colocar textos, imagens, vídeos, links e entre outros.

CSS

Cascading Style Sheets

Nós utilizamos o CSS para estilizar a página web, para definir como um conteúdo vai aparecer, como por exemplo: o alinhamento de um conteúdo, fonte de algum texto, entre outros.

JavaScript

Nós utilizamos o JavaScript para a página deixando de ser estática, baseando-se em protótipos, multi-paradigma e dinâmica, suportando estilos e adicionando animações.

Tags semânticas

As tags semânticas são utilizadas para fornecer um significado de um conteúdo, deixando o código mais legível e melhorando o entendimento do conteúdo.

DOM

Document Object Model

O DOM é utilizado para representar a estrutura de um documento como uma árvore de nós, onde cada nó é um objeto representando uma parte do documento


Comandos/Tags:

  • header // Suporte introdutório ou navegacional;
  • nav // Seção que aponta para outras páginas;
  • main // Define conteúdo principal da página;
  • section // Seção genérica;
  • footer // Representa o rodapé;
  • a // Descreve um link.
  • var element = document.getElementById('myElementId') // Seleciona um elemento pelo seu ID;
  • var elements = document.getElementsByClassName('myClassName') // Seleciona todos os elementos com uma classe específica;
  • var elements = document.getElementsByTagName('div') // Seleciona todos os elementos com uma tag específica;
  • var element = document.querySelector('.myClassName') // Seleciona o primeiro elemento que corresponde a um seletor CSS;
  • var elements = document.querySelectorAll('.myClassName') // Seleciona todos os elementos que correspondem a um seletor CSS;
  • var parent = element.parentNode // Acessar o elemento pai.
  • var children = element.childNodes // Acessar os elementos filhos.
